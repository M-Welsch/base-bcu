The serial communication required for flashing the SBC cannot be established for some reason inside BaSe.
To flash a new firmware you have to follow these steps:
1. Stop BaSe (here)
2. Log on your BaSe via your SSH Client 3. change to the folder /home/maxi/base/sbc_interface/
4. run sudo python3 sbc_updater.py
5. restart BaSe
